# HPC_Project3_RandomMatrixTheory
Project 3: Using MPI to compute maximal eigenvalues of large GOE matrices in order to approximate the Tracy-Widom distribution

# Run Scripts
The `runScript.sh` script will run the full suite of tests performed for the paper. This takes a very long time. 
For a quicker demo of operation, use the `runTest.sh` script, which uses smaller matrices and ensembles in order to display functionality.
